# 概要
オプション画面

# コンポーネント
* SE音量スライダー
* BGM音量スライダー
* プロフィール変更ボタン
* 成績ボタン
* 戻るボタン

## SE音量スライダー
SEの音量を変更するスライダー  
指定範囲は0～100%  

## BGM音量スライダー
BGMの音量を変更するスライダー  
指定範囲は0～100%

## プロフィール変更ボタン
プロフィール設定画面を開く

## 成績ボタン
成績画面を開く

## 戻るボタン
前の画面に戻る
