# 概要
ユーザ名を入力するダイアログ

# コンポーネント
* 概要説明ラベル
* 名前入力テキストボックス
* OKボタン
* 戻るボタン

## 概要説明ラベル
ユーザに名前を入力することを促すラベル  
「名前を入力してください(後からオプションで変更できます)」

## 名前入力テキストボックス
名前入力欄  
オプションから画面が開かれた際には、現在のユーザ名を表示する  

## OKボタン
入力された名前で決定する  
名前が入力されていない場合は非活性とする

## 戻るボタン
変更をキャンセルする  
オプションから画面が開かれた場合のみ表示する

# 詳細
アプリ起動時とオプションからユーザ名を変更する際に生成される
